Web Mapping with IUENNA Project

Overview

This straightforward web mapping project demonstrates how to easily create a web mapping applications (WMAs) using QGIS, the qgis2web plugin, and Generative Artificial Intelligence (GenAI). WMAs are powerful tools for visualizing spatial data, also in the field of archaeology and in the case one wants to make archaeological data more accessible and individually manageable easily. By integrating maps with interactive features, WMAs enable researchers and the public to access, explore, and analyze even vast collections of archaeological data. In the context of IUENNA, WMA may serve as interfaces to navigate archaeological records from the Jauntal region for specific tasks like getting a general impression of the dataset or, alternatively, search for a single ressource or even a specific group of objects within this dataset, thereby complementing the ARCHE repository's structured data access by providing addional layers for the user, easily customable and even individually generable for specific tasks to easy dayly work. This integration enhances usability and democratizes access to cultural heritage by shifitng access from the analog object, only avalable as closed access at a specific point in space and time ususally for a single individual at a dedicated storage place or by providing dispersed data on various data carriers to global structured open access enhanced with meta data for all interested users by the means of didigal ubiquity, futhermore reducing mechanical stress to the analog objects.

One of the key components in this sense can be the use of open-source JavaScript library for interactive maps like LINK Open Layers or Leaflet. Such libraries enable the addition of various map layers, markers, and geometries, making it a powerful tool for web mapping applications.

Steps to Create the Web Mapping Application

  1. Install QGIS on your computer (Official QGIS Website).
  2. Prepare your data in QGIS, e.g., by adding vector data (Shapefiles) or raster data.
  3. Install the qgis2web plugin via the QGIS Plugin Manager.
  4. Export your project as a web map using qgis2web. Select options such as Leaflet or OpenLayers.
  5. Copy the exported files (HTML, CSS, JavaScript) to your GitHub repository.
  6. Enable GitHub Pages for your repository to make the map available online.

Example Code

The following HTML code shows a minimalist web mapping integration using Leaflet:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Example Map</title>
    <link rel="stylesheet" href="https://unpkg.com/leaflet/dist/leaflet.css" />
    <script src="https://unpkg.com/leaflet/dist/leaflet.js"></script>
</head>
<body>
    <div id="map" style="width: 100%; height: 500px;"></div>
    <script>
        var map = L.map('map').setView([46.5647, 14.8222], 13);
        L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png').addTo(map);
    </script>
</body>
</html>

Using Generative AI Tools

Generative AI tools like ChatGPT, Claude, and Gemini have revolutionized the way we create and manage digital content. These tools provide invaluable assistance in setting up web pages, offering suggestions, generating code, and refining textual content. For example, Claude excels at generating detailed documents, while Gemini focuses on multimodal tasks, integrating text and images. ChatGPT is widely used for versatile coding and text editing tasks.

Generative AI enhances productivity by:

  • Time Efficiency: Automating repetitive coding tasks, allowing developers to focus on more complex challenges.
  • Accessibility: Lowering the entry barrier for beginners by offering step-by-step guidance and simplifying technical concepts.
  • Error Reduction: Providing boilerplate code and recommendations that align with best practices, reducing debugging time.
  • Customization: Tailoring suggestions and templates to specific needs, saving hours in project-specific implementations.
  • Versatility: Supporting a wide array of tasks beyond web development, from content generation to data visualization.

Generative AI tools are not just productivity enhancers but also democratize digital innovation, enabling users with minimal technical background to participate effectively in creating and sharing impactful projects.
