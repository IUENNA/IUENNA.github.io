Web Mapping with QGIS – IUENNA Project

Overview

This simple web mapping project demonstrates how to create a web mapping application with QGIS and the qgis2web plugin. The aim is to make long-term archival data more accessible and individually manageable. Web Mapping Applications (WMAs) are powerful tools for visualizing spatial data, especially in the field of archaeology. By integrating maps with interactive features, WMAs enable researchers and the public to access, explore, and analyze vast collections of archaeological data. In the context of IUENNA, a WMA serves as an interface to navigate archaeological records from the Jauntal region, complementing the ARCHE repository's structured data access. This integration enhances usability and democratizes access to cultural heritage.

One of the key components of this project is the use of Leaflet, an open-source JavaScript library for interactive maps. Leaflet enables the addition of various map layers, markers, and geometries, making it a powerful tool for web mapping applications.

Steps to Create the Web Mapping Application

  1. Install QGIS on your computer (Official QGIS Website).
  2. Prepare your data in QGIS, e.g., by adding vector data (Shapefiles) or raster data.
  3. Install the qgis2web plugin via the QGIS Plugin Manager.
  4. Export your project as a web map using qgis2web. Select options such as Leaflet or OpenLayers.
  5. Copy the exported files (HTML, CSS, JavaScript) to your GitHub repository.
  6. Enable GitHub Pages for your repository to make the map available online.

Example Code

The following HTML code shows a minimalist web mapping integration using Leaflet:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Example Map</title>
    <link rel="stylesheet" href="https://unpkg.com/leaflet/dist/leaflet.css" />
    <script src="https://unpkg.com/leaflet/dist/leaflet.js"></script>
</head>
<body>
    <div id="map" style="width: 100%; height: 500px;"></div>
    <script>
        var map = L.map('map').setView([46.5647, 14.8222], 13);
        L.tileLayer('https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png').addTo(map);
    </script>
</body>
</html>
